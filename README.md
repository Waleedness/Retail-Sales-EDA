# Retail Sales Exploratory Data Analysis (EDA) Using Python

## Project Overview
This project performs an exploratory data analysis on a retail sales dataset to better understand business performance, customer behavior, product trends, and store performance. The goal is to transform raw transactional data into meaningful insights that could help guide business decisions.

The analysis focuses on revenue metrics, customer activity, product category performance, location performance, and sales trends over time. Visualizations were created to clearly highlight patterns in the data.

---

## Tools & Technologies
- Python  
- Pandas (data analysis and manipulation)  
- Matplotlib (data visualization)  
- Excel (data source)

---

## Dataset Overview
The dataset contains retail transaction data including:

- Transaction ID (TID)
- Customer ID (CID)
- Purchase Date
- Store Location
- Product Category
- Gross Amount
- Net Amount

These fields allow analysis of revenue, discounts, customer purchasing behavior, and store performance.

---

## Key Analysis Performed

### Revenue Analysis
- Total gross and net revenue
- Total discounts given

### Transaction Analysis
- Total number of transactions
- Average transaction value

### Customer Analysis
- Total unique customers
- Average revenue per customer
- Top customers by revenue
- Customer purchase frequency

### Product Category Performance
- Revenue and transaction count by product category
- Identification of best-performing categories

### Store Location Performance
- Revenue and average transaction value by location
- Identification of top and underperforming stores

### Time-Based Revenue Trends
- Revenue trends by year and month
- Identification of seasonal peaks and patterns

### Revenue Distribution
- Histogram and boxplot of transaction values
- Detection of outliers and negative transactions

### Discount Analysis
- Total discounts by product category
- Understanding impact on revenue

---

## Key Insights

1. **Revenue Drivers:** A small number of product categories contribute significantly to total revenue.  
2. **Customer Contribution:** A portion of customers generates a large share of revenue, highlighting the importance of retention.  
3. **Store Performance:** Some store locations outperform others, suggesting differences in demand or operations.  
4. **Seasonal Trends:** Certain months show higher revenue, indicating opportunities for targeted promotions.  
5. **Discount Impact:** Excessive discounts in some categories reduce profitability.

---

## Business Recommendations

1. Focus marketing on high-performing product categories.  
2. Investigate underperforming store locations to identify improvement opportunities.  
3. Develop customer retention strategies to increase repeat purchases.  
4. Optimize discount strategies to balance revenue and profitability.  
5. Align promotional campaigns with seasonal demand patterns.

This project demonstrates how Python can be used to transform raw retail transaction data into actionable business insights through exploratory data analysis and visualization.
